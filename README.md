# AI Task Analyzer - Chrome拡張機能

ブラウザでの操作内容を記録し、AIでAI代替可能なタスクを提案するChrome拡張機能です。

## 🎯 機能

- **自動記録**: ブラウザ操作を詳細に記録（URL訪問、クリック、フォーム入力、コピペなど）
- **AI分析**: OpenAI APIを使用して、自動化可能なタスクを特定
- **定時通知**: 毎日18時（設定変更可能）に分析結果を通知
- **プライバシー保護**: パスワードや機密情報は記録されません

## 📦 インストール方法

### 1. 必要なもの

- Google Chrome ブラウザ
- OpenAI API Key（[こちら](https://platform.openai.com/api-keys)から取得）

### 2. インストール手順

1. この拡張機能のフォルダ（`ai-task-analyzer-extension`）を任意の場所に配置

2. Chromeブラウザで `chrome://extensions/` を開く

3. 右上の「デベロッパーモード」をONにする

4. 「パッケージ化されていない拡張機能を読み込む」をクリック

5. `ai-task-analyzer-extension` フォルダを選択

6. 拡張機能がインストールされます（Chromeのデフォルトアイコンが表示されます）

## ⚙️ 設定

1. 拡張機能アイコンをクリックしてポップアップを開く

2. 「設定」タブを選択

3. OpenAI API Keyを入力

4. 通知時刻を設定（デフォルト: 18:00）

5. 「設定を保存」をクリック

## 📊 使い方

### 自動分析

- 設定が完了すると、ブラウザ操作が自動的に記録されます
- 毎日設定した時刻に自動でAI分析が実行されます
- 通知が表示されたら、クリックして結果を確認できます

### 手動分析

1. 拡張機能アイコンをクリック
2. 「ダッシュボード」タブで「今すぐ分析」をクリック
3. 分析が完了すると「分析結果」タブに結果が表示されます

## 🔍 分析内容

AIは以下の観点から分析を行います：

1. **繰り返し作業**: 同じサイトへの頻繁なアクセス
2. **データ入力**: フォーム入力や検索操作
3. **情報収集**: 複数サイトからの情報収集
4. **コピー＆ペースト**: データの転記作業
5. **プロダクト提案**: 自動化プロダクトのアイデア

分析結果には以下が含まれます：

- **自動化可能なタスク一覧**
  - タスクの説明
  - 頻度と優先度
  - 自動化の方法
  - 推定削減時間

- **プロダクトアイデア**
  - プロダクト名と説明
  - 対象タスク

- **総評と推奨事項**

## 🔒 プライバシーとセキュリティ

### 記録されないデータ

- パスワード入力フィールド
- クレジットカード情報
- chrome:// や chrome-extension:// ページ

### データの保存

- 記録データは**ブラウザ内（Chrome Storage）のみ**に保存されます
- 外部サーバーには送信されません（OpenAI API分析時を除く）

### OpenAI APIへの送信

- 分析実行時のみ、サマリー化されたデータを送信
- 機密情報は除外されます
- OpenAIのプライバシーポリシーに準拠

## 📈 データ管理

### データの確認

「ダッシュボード」タブで以下を確認できます：
- 総アクティビティ数
- 訪問サイト数
- 記録状態

### データのクリア

「ダッシュボード」タブの「データをクリア」ボタンで、本日の記録を削除できます。

### 記録の停止

「設定」タブで「記録を有効にする」をOFFにすると、記録を一時停止できます。

## 🛠️ トラブルシューティング

### 分析が実行されない

1. OpenAI API Keyが正しく設定されているか確認
2. API Keyに十分なクレジットがあるか確認
3. ブラウザのコンソールでエラーを確認

### 通知が表示されない

1. Chromeの通知設定を確認
2. 拡張機能の通知権限を確認
3. アラームが正しく設定されているか確認（background.jsのログ）

### データが記録されない

1. 「記録を有効にする」がONになっているか確認
2. 拡張機能が有効になっているか確認
3. コンソールでエラーを確認

### デバッグ方法

**Background Service Worker**:
1. `chrome://extensions/` を開く
2. 拡張機能の「サービスワーカー」をクリック
3. コンソールでログを確認

**Content Script**:
1. 任意のページでF12を押して開発者ツールを開く
2. コンソールタブでログを確認

**Popup**:
1. ポップアップを開いた状態で右クリック → 「検証」
2. 開発者ツールでデバッグ

## 📝 カスタマイズ

### 分析プロンプトの変更

`background.js` の `analyzeWithOpenAI` 関数内のプロンプトを編集することで、分析内容をカスタマイズできます。

### 記録するイベントの追加

`content.js` にイベントリスナーを追加することで、記録する操作を増やせます。

### 通知時刻の変更

設定画面から変更可能です。または、`background.js` の `setupDailyAlarm` 関数を編集してください。
